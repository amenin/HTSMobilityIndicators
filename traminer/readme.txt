How to use these files:

1. Create the activity and mode-based sequences
  - run prepare_sequences.r
  
2. Compute the Distance Matrix and the Dendogram 
  - run classification.r -area "name of the HTS' area"

3. (optional) Generate the charts to visualize the classes from the dendogram
  - run charts.r
